# Beginner part-time agent compensation.docx

Beginner Part-Time agent compensation 

Compensation Expectations for Brand New Agents working part-time at 10 hours per week. 

Starting Commission:

50% starting rate

Average Case:

$150/month premium = $1,800 commissionable premium per case

Monthly Production:

2 cases/month → $1,800 × 2 = $3,200 commissionable premium

Monthly Earnings:

$3,200 × 50% = $1,800 commissionable

Carrier advance (75%) = $1,350 / upfront

Total Monthly Compensation:

$1,350
